# /recommend - 个性化推荐

根据用户风格生成个性化股票推荐。

## 使用方式

```
/recommend                    # 生成个性化推荐
/recommend --limit 5          # 限制返回数量
/recommend --style swing      # 指定交易风格
/recommend --risk aggressive  # 指定风险偏好
```

## 示例

```
/recommend                    # 使用默认配置推荐
/recommend --limit 5          # 返回前5只股票
/recommend --style day_trading --limit 10  # 日内交易风格推荐
```

## 输出格式

```
个性化推荐 (基于您的交易风格: 波段交易)
┌──────────────────────────────────────────────────────────────────┐
│  代码     名称          得分    风格匹配  推荐策略              │
├──────────────────────────────────────────────────────────────────┤
│  000001   平安银行      3.5     85%       波段交易, 回调买入    │
│  600519   贵州茅台      2.8     72%       波段交易, 趋势跟随    │
└──────────────────────────────────────────────────────────────────┘
```

## 推荐因子

系统根据您的交易风格自动选择合适的选股因子：

| 交易风格 | 重点因子 |
|----------|----------|
| 日内交易 | 动量、波动率 |
| 波段交易 | 动量、质量、估值 |
| 趋势跟踪 | 动量、质量 |
| 价值投资 | 估值、质量 |
